Question-1: What are the different ways to select an element in the DOM?

There are several ways to select elements in the DOM using JavaScript:
1. `getElementById("id")` – Selects an element by its ID.
2. `getElementsByClassName("class")` – Selects elements by class name (returns a collection).
3. `getElementsByTagName("tag")` – Selects elements by tag name (collection).
4. `querySelector("selector")` – Selects the first matching element using CSS selectors.
5. `querySelectorAll("selector")` – Selects all matching elements (NodeList).

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Difference between `innerHTML`, `innerText`, and `textContent`:
1. `innerHTML` – Gets or sets HTML content, including tags.
2. `innerText` – Gets or sets only visible text, ignoring hidden elements.
3. `textContent`– Gets or sets all text, including hidden content, without parsing HTML.
